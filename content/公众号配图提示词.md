# 公众号配图提示词清单

> 用途：发布公众号文章时需要的封面/插图。
> 工具：腾讯元宝（imagen.baidu.com / yuanbao.tencent.com）或即梦 AI（jimeng.jianying.com）。

---

## 📸 图片清单

### 1. 公众号头像（圆/方）

**提示词**：
```
简约科技风 Logo，"丰"字艺术化设计，深蓝渐变背景
主色 #0066ff + #00aaff，扁平化设计，可识别性强
1080×1080px，公众号头像专用，圆角方形预览下都好看
```

### 2. 公众号封面（首篇文章）

**提示词**：
```
一名全栈开发者在咖啡馆写代码，宝鸡的城市天际线作为背景窗户
深色科技风，屏幕上有代码、图表、AI 助手图标
"一人技术公司"主题，明亮、专业、有故事感
900×383px（公众号封面推荐尺寸）
```

### 3. 内文插图 1：AI 工作流

**提示词**：
```
扁平化插图：屏幕中央显示代码编辑器，周围环绕 6 个小图标
（机器人 / 数据流 / 云端 / 文档 / 图表 / 用户）
蓝色渐变背景 #0066ff → #00aaff
表达"AI 自动化提效"主题，简约现代
```

### 4. 内文插图 2：一人公司流程

**提示词**：
```
流程图插图：1 个开发者 → 6 个环节（需求 → 设计 → 开发 → 测试 → 部署 → 运维）
每个环节用不同图标表示，节点之间有箭头连接
蓝色科技风，背景渐变 #0a0e27 → #0066ff
中心开发者图像带光环，表达"超级个体"主题
```

---

### 5. 案例展示封面图

**提示词**：
```
6 个卡片式图标展示 6 种服务：CRM / 官网 / AI / 小程序 / 部署 / 自动化
每个图标采用 3D 风格，蓝色调为主，配色统一
背景白色或浅灰，专业、整齐、有秩序感
1200×675px（适合分享到朋友圈）
```

---

## 🎯 推荐生成顺序

1. **先生成头像**（用腾讯元宝的"logo 设计"模式，3 分钟）
2. **再生成首篇文章封面**（明确指定尺寸）
3. **内文插图按需生成**（一次生成多个，挑最好的）

## 📐 尺寸参考

| 用途 | 尺寸 |
|---|---|
| 公众号头像 | 1080×1080 |
| 文章封面 | 900×383（首）/ 1080×607（次） |
| 内文插图 | 1080×608（横版）/ 750×750（方版） |
| 朋友圈分享 | 1200×675 |

## 💾 保存路径

生成后下载到 `content/images/` 目录，公众号文章里使用微信素材库即可。

```bash
mkdir -p content/images
# 下载图片后移动到这里
```
