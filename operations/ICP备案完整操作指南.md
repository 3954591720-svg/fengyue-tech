- **可能电话回访法人/负责人** ← 接电话！漏接直接驳回
- 通过：收到「陕 ICP 备 XXXXXX 号」备案号
- 驳回：写明问题，修改后二次提交

## 五、高频驳回避坑要点

| 坑点 | 后果 | 怎么避 |
|---|---|---|
| 域名实名主体 ≠ 公司名 | 直接驳回 | 域名实名信息必须 = 公司营业执照 |
| 网站名称含敏感词 | 直接驳回 | 看我准备的 3 个备选名 |
| 证件照片模糊/反光 | 退回重传 | 拍照时四角完整、无反光 |
| 手机号非本人实名 | 直接驳回 | 用你本人身份证办的号 |
| 漏接管局回访 | 直接驳回 | 全程保持开机、接陌生电话 |
| 服务器为境外主机 | 不能备案 | 用国内轻量云/ECS |
| 未获取备案服务码 | 不能备案 | 买服务器时勾选「备案用」 |
| 备案完成 30 天未做公安备案 | 域名被关停 | 30 日内必须做 |

## 六、公安联网备案（备案通过后必办）

⏳**你自己办理**（需法人注册账号 + 实人认证）

### 流程
1. 注册：全国互联网安全管理服务平台 https://beian.gov.cn
2. 企业法人注册账号
3. 完成实人认证
4. 录入：企业信息、域名、ICP 备案号、服务器 IP
5. 上传：营业执照、ICP 备案截图、首页截图
6. 提交审核
7. 通过 → 拿到公安备案编号，放网站底部
